# 🌍 EC Token RS

Plataforma gamificada para promover ações sustentáveis em Porto Alegre, RS.

## ✨ Características

- 📊 **Dashboard** - Estatísticas de impacto em tempo real
- 🗺️ **Mapa** - Ações sustentáveis na cidade
- 👤 **Meu Impacto** - Perfil pessoal e histórico
- 🔔 **Feed** - Notificações da comunidade
- 🛍️ **Loja** - Resgate de tokens por produtos

## 🚀 Como Usar

### Local
```bash
# 1. Instale as dependências
pip install -r requirements.txt

# 2. Execute
streamlit run app.py

# 3. Abra (automático)
http://localhost:8501
```

### Hugging Face Spaces

1. Vá para: `huggingface.co/spaces`
2. Clique: `Create new Space`
3. SDK: `Docker`
4. Faça upload dos arquivos:
   - `app.py`
   - `requirements.txt`
   - `Dockerfile`
   - `README.md`
5. Aguarde 3-5 minutos
